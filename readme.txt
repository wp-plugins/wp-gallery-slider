The plugin automatically add the highslide-image class to linked images in a post.

== Description ==
This plugin automatically add the Highslide onclick="return hs.expand(this);" to linked images  in a post, focusing on the wp-gallery shortcode. The required Highslide for WP files are Already Added so dont worry. Just Activate the plugin.


== Installation ==

This section describes how to install the plugin and get it working.

1. Install through the WordPress Plugin directory at your WordPress site backend.
2. Activate the plugin through the 'Plugins' menu in WordPress Admin.
3. Create Your photo galley as usual.
4. Preview images at front ends.

== Usage ==
After activation of the Plugin, just create your wp-gallery via the media library.
Job Done! 
And when you click on any Thumbnail in the gallery it will pop up.
 
On mouse over control shows, you can **PLAY** **PAUSE** **DRAG** **NEXT** **PREV** **CLOSE** **FULLSCREEN**.

Even use your keyboard to control the slide.

**Will this plugin conflict with another plugin**

I am not sure but try to disable all other plugin(s) that might control or modifies your gallery.
One of them is JetPack gallery, PrettyPhoto, although I have try to remove any attribute value that may cause conflict
